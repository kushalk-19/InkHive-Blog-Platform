# Inkhive - AI Powered Blog Platform

A full-stack blog platform where users can create blogs with AI assistance, admins approve content, and everyone can give feedback.

## Features
- AI blog generation using Google Gemini
- User registration, profile, and image upload
- Admin dashboard to manage users & blogs
- Responsive design

## Tech Stack
**Frontend:** React + Vite + Tailwind CSS + ShadCN/UI  
**Backend:** Node.js + Express + MongoDB + Mongoose  
**AI:** Google Gemini API  
**Authentication:** JWT with secure cookies

## How to Run Locally
1. Clone the repo
2. Backend: `cd backend && npm install && npm start`
3. Frontend: `cd frontend && npm install && npm run dev`

## Notes
1) Inside frontend folder in .env file put your Gemini API key.
2) Inside backend folder in .env file put your JWT secret key and MongoDB URI
